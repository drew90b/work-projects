# Inbox

Everything lands here first as entries in `inbox.md` — Telegram messages, verbal requests,
paper notes, anything not immediately actionable. Process `inbox.md` each morning: read
each item, decide what it is, route it to the right folder (`boss-requests/`,
`maintenance-requests/active/`, `admin-requests/`, `personnel/`, or `projects/`), and
remove or strike the entry once handled. The goal is inbox zero before 9 AM.
